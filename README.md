# 🪲 Aguia Dedetização

> Página institucional desenvolvida para um cliente real do ramo de dedetização e controle de pragas.

## 📋 Descrição

Site criado para apresentar a empresa Aguia Dedetização ao público, com foco em transmitir profissionalismo, destacar os serviços oferecidos e facilitar o contato com o cliente.

## ⚙️ O que o site faz

- Apresenta a empresa e seus diferenciais
- Lista os serviços de dedetização disponíveis
- Exibe informações de contato e localização
- Página de apresentação para cliente real
