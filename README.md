# 🪲 Águia Dedetizadora - Grupo Preventec

> Site institucional otimizado para SEO com foco em posicionamento orgânico no Google.

## 📋 Descrição

Site criado para apresentar a empresa Águia Dedetizadora ao público, com foco em transmitir profissionalismo, destacar os serviços oferecidos e facilitar o contato com o cliente.

**URL**: https://aguiadedetizadora.com.br/

## ⚙️ Funcionalidades

- Apresentação da empresa e diferenciais
- Lista completa de serviços de dedetização
- FAQ com 7 perguntas frequentes (rico para snippets do Google)
- Seção de áreas de atendimento (Grande São Paulo)
- Conteúdo educativo sobre controle de pragas
- Botão flutuante do WhatsApp
- Design responsivo (mobile-first)

## 📈 SEO Implementado

### On-Page
- ✅ Title tag otimizada: "Dedetizadora em São Paulo 24h | Águia Dedetização"
- ✅ Meta description com CTA e palavras-chave
- ✅ Schema.org LocalBusiness + FAQPage (JSON-LD)
- ✅ Sitemap XML com 6 URLs
- ✅ Headings hierárquicos (H1, H2, H3)
- ✅ Conteúdo estratégico para dúvidas frequentes
- ✅ Links internos entre seções

### Palavras-chave alvo
- "dedetizadora em são paulo"
- "dedetização zona leste", "dedetizadora zona sul"
- "quanto custa dedetização"
- "dedetização é segura para pets"
- "com que frequência dedetizar"
